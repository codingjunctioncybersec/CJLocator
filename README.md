# CJLocator

CJLocator is a Python-based tool developed by Coding Junction.

## Ethical Use

CJLocator is a Python-based tool that demonstrates geolocation request workflows and can be used to collect location information from users **only when they have explicitly granted location permission**.

This project is intended for educational purposes, security research, and authorized testing only.

Use this tool only with the explicit permission of the owner or in environments where you are authorized to perform testing. The developers are not responsible for any misuse, unauthorized data collection, or illegal activities carried out using this software.

## Features

- Custom templates
- Telegram Support
- WhatsApp Support
- News Template
- Custom OG Tags
- Discord Webhook Support

## Installation

```bash
git clone https://github.com/codingjunctioncybersec/CJLocator.git
cd CJLocator
bash install.sh
python3 CJLocator.py
```
